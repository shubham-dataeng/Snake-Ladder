# Snake & Ladder (C Project)

## 🎯 Description

A console-based Snake & Ladder game built in C with modular design.

## 🚀 Features

* Multiplayer support
* Dice roll simulation
* Snakes & ladders logic
* Clean modular code structure

## 🛠️ How to Run

```bash
make
make run
make debug
make memcheck
make clean
```

## 📁 Project Structure

* src/ → source files
* include/ → header files
* data/ → configurations
* tests/ → test cases
